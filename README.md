## 📖 Sobre

O **Relógio Digital** é um projeto básico de front-end desenvolvido com **HTML**, **CSS**, e **JavaScript**. Ele utiliza funcionalidades fundamentais dessas tecnologias para criar um relógio funcional, dinâmico e visualmente agradável.

### 🚀 Aprendizados
- Manipulação do DOM com **JavaScript**.
- Estilização responsiva e minimalista com **CSS**.
- Estruturação de um projeto básico com **HTML**.
